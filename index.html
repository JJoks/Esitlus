What are AJAX Callback Commands

Overview

If an AJAX request was a Big Mac, Callback commands would be the secret sauce.

What does that mean?

  • Functions returned and invoked by all AJAX requests
  • Defined by the AJAX API
    • Javascript functions
    • PHP classes
  • Defined by custom modules
  • Provide PHP ability to invoke JS functions

Client side: Javascript

A function attached to a global object

Accepts three arguments

  • Ajax
  • Response
  • status

A wrapper for other javascript

Server side: PHP

A class that impletement the CommandInterface

Must have a method called 'render'

  • Returns an associative array
  • Must have element with key of 'command'
