math-katex plugin

  • Uses KaTeX for lightweight rendering
  • Renders all formulas when presentation loads
    (no re-rendering when slide changes)
  • No MathJax CDN dependency

Two usage modes

  1. TeX-style, wrapped in $…$ or $$…$$ (escape with \$). Works in Markdown, too. Get a class formula for styles
  2. Wrapped in HTML elements with class formula (or math)

Tex style (dollars)

\$\$\frac{1}{1 + e^{-x}}\$\$

for example yields a display formula (block):

$$ \frac{1}{1 + e^{-x}} $$

Single \$…\$ render in-line: $-\sum_{i=1}^n p(x_i) \log p(x_i)$.

Wrapped in an element

Wrap the formula in an element
with class formula (or math): 

<div class="formula">\frac{1}{1 + e^{-x}}</div>
renders as:
\frac{1}{1 + e^{-x}}
Or in-line, with a <span>: P(A \mid B)
