Week 1: Layout and Spacing

notes

1. Start with too much white space

It's easier to remove white space than to add it to an already cramped design

2. You don't have to fill the whole screen

Spreading things too wide makes an interface harder to interpret

If you only need 600px, use 600px

3. Avoid ambiguous spacing

Include clear separation between UI elements

Don't forget about horizontal spacing!
